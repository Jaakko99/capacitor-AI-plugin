# Capacitor Recolor Plugin
Native on-device object segmentation and transformation for Capacitor apps.
Bring real-time AI image editing to your hybrid mobile applications using CoreML (iOS) and ML Kit (Android). No cloud processing. No external APIs. Full privacy.

# Why It Exists
Capacitor apps often lack direct access to native AI capabilities. This plugin bridges that gap, enabling high-performance object detection and selective image transformations directly on the device.
Product visualization, interior design previews, virtual clothing try-ons.

# Core Capabilities 
1) Automatic object segmentation (furniture, clothing, vehicles, etc.)
2) Selective recoloring while preserving shading and texture
3) Real-time performance optimized for mobile
4) Fully offline processing
5) No image data leaves the device

# Example Use Cases 
1) Furniture visualization (change sofa colors instantly)
2) Fashion apps (preview clothing color variants)
3) Interior design tools (test wall or decor colors)
4) E-commerce product customization
