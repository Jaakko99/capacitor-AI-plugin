# recolor-plugin

Native on-device object segmentation and transformation for Capacitor apps

## Install

To use npm

```bash
npm install recolor-plugin
````

To use yarn

```bash
yarn add recolor-plugin
```

Sync native files

```bash
npx cap sync
```
